
# SmartRecruit

SmartRecruit is a Laravel-based application for intelligent employee recruitment using Fuzzy Logic methodology.

## Overview

This project implements a **Fuzzy Logic-based decision support system (SPK)** for evaluating and selecting new employees based on multiple criteria.
